## Objetivo

El objetivo de este trabajo es analizar la relación entre el precio del combustible (jet fuel) y las tarifas aéreas, con el fin de determinar en qué medida los costos energéticos explican la variación en el precio final de los pasajes. Se busca identificar patrones, correlaciones y posibles factores adicionales que intervienen en la formación de precios.

## Dataset

El dataset utilizado fue extraído de Kaggle, y contiene información sobre tarifas aéreas (tarifa base, impuestos, recargo por combustible y tarifa total), junto con variables del mercado energético como el precio del crudo Brent y del jet fuel. 

Fuente:
- Dataset: (https://www.kaggle.com/datasets)

## Metodología

El análisis se desarrolló en dos etapas complementarias:

1. **Exploración de datos en Python (Google Colab)**:
   - Limpieza y transformación del dataset
   - Detección de outliers
   - Análisis de correlación entre variabilidad de precios y fases geopolíticas 
   - Conclusiones
     
2. **Visualización en Power BI**:
   - Elaboración de dashboards 
   - Análisis comparativo entre tarifas y precios del combustible
   - Segmentación por fases geopolíticas y regiones
